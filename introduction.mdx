## Introduction

 This guide provides comprehensive documentation for integrating with the Niobi Payments API, allowing your users to make collections and payouts through various payment channels to individuals and businesses. The Niobi Payments API enables you to manage wallets, top-up funds, and initiate payouts seamlessly.

## API Overview

The Niobi Payments API has the following key components:

- ***Wallets:*** Create and manage wallets to store funds for payouts. Retrieve wallet details and monitor available balances. 
- ***Top-up Funds:*** Initiate top-up requests to add funds to wallets. Also, utilize payment links to collect payments from customers. 
- ***Initiate Payouts:*** Facilitate payouts from wallets to recipients. Initiate payouts through various payment channels, such as bank transfers or digital wallets.
