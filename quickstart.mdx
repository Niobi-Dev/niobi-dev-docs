## Setup your Environment

Niobi provides both the sandbox and live Environments.


<AccordionGroup>
  <Accordion icon="github" title="Sandbox">
    During the development process, we created a sandbox envrironment where you can easily interact our APIs without making real transactions.
    To get started, follow the signup instructions using this link
    [sandbox](https://dev.users.niobi.co).
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Live">
    After testing with the sandbox and ready for production.Follow these steps. 1. Follow the signup instructions using this link
    [Live](https://app.niobi.co).
    2. Change the API endpoints from 
    `https://dev.users.niobi.co` to  ```https://users.niobi.co```
  </Accordion>
</AccordionGroup>
